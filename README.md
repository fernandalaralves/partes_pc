# Projeto Partes do PC

## Descrição:

Esse site bastante simples, visa ensinar de maneira divertida
crianças dos 6 aos 12 anos conceitos básicos sobre o hardware
de um computador.

---

## Como funciona?

A criança deverá ler e se atentar as descrições de cada componente,
após concluir sua leitura, o responsável ou professor deverá indusi-lá a
clicar no "Quiz" onde irá responder perguntas sobre o que foi lido.
Se acertar mais de 12 questões, a criança poderá escolher entre o
minesweeper (campo minado) ou snake game (jogo da cobrinha).
